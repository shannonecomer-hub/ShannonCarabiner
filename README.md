# Shannon Carabiner

An interactive 3D carabiner visualization with clickable tags and links.

## Features

- Interactive 3D carabiner graphic with realistic depth and shadows
- Clickable tags positioned around the carabiner
- Smooth animations and hover effects
- Responsive design

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Tech Stack

- React 19
- Vite
- CSS3 with 3D transforms
